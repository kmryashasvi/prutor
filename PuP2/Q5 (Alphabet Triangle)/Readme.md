# Alphabet Triangle
## LAB2
## PROBLEM STATEMENT
Write a program to print the Alphabet pattern, as shown below. The input will be a number n, such that 1 <= n <= 26.

### NOTE: 
1. There is a single ```_``` (underscore) at the start, at the end, and between two letters on each line.

### Examples
Input: 3<br>
Output: <br>
```
_A_
_A_B_
_A_B_C_
```
<br>
Input: 4<br>
Output: <br>

```
_A_
_A_B_
_A_B_C_
_A_B_C_D_
```
